Card Frame

Welcome Tutorial

Upload cropped and upscaled official artwork (required, JPG/PNG supported).

Enter Character name, Series title (Simplify if needed, only English titles will be used, e.g. Danganronpa V3: Killing Harmony -> Danganronpa) and Print number (Optional).

Use the auto generated color palette, or select a Custom Color for the frame. (Use the sliders, Supports Hex/RGB too)

Click the Download PNG button at the top right of the website to save your finished card image.

Tip: If the first automatic dominant color choice looks off, pick a different palette color and reclick the first color palette and it will show a different color that may be better, if it isn't try using the other generated colors or customize the color's yourself (you can use https://imagecolorpicker.com) or reupload to restore a previous dominant color.
