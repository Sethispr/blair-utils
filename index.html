Card Frame

Welcome Tutorial

Begin by uploading your artwork (required, JPG/PNG supported).

Input the Character Name and their Series Title. You do not need to add a print number if you are submitting a card.

The card frame color is automatically detected from your image. You can customize this by selecting a different color from the color palette or choosing a Custom Color.

Once your done, click the "Download PNG" button (top right of the website) to save your finished card image.

Tip: If the automatic color choice looks off, pick a different color palette and switch back to the first color palette (it'll change the color to sth else). If the previous color was better, reupload to restore a previous dominant color.

Lossless Compression
  • Note: You do not need to compress an image for us, we will do it for you! This is completely optional.
  • Blair Utils (needs Rust/Python, uses Oxipng, has batch support)
  • TinyPNG (Upload final card image, press blue "Edit" button, select Oxipng, set compression level 3, interlace off, has no batch support)
  • Squoosh (select Oxipng, BUGGED)
